Address Autocomplete Setup SETUP

Two options below — pick whichever suits you. Both work fine; Option A just gives live address suggestions as you type.

Option A — Recommended
Google Places Autocomplete
  • Type an address → see live suggestions
  • Auto-fills suburb & postcode on pick
  • Australian addresses only
  • Free (up to ~10,000 lookups/month)
  • Requires a free Google API key (~2 min setup)
Option B — Already Active
Manual Entry
  • Type address, suburb, postcode manually
  • Works right now, no setup needed
  • No internet dependency
  • No API key required
  • Fields are already on the form
Option A Setup — Google API Key 5 MIN
1

Go to Google Cloud Console

Visit console.cloud.google.com and sign in with any Google account (your personal Gmail is fine).

2

Create a Project & Enable Places API

Create a new project (e.g. "TJS Quotes"), then go to APIs & Services → Library, search for Places API and enable it.

3

Create an API Key

Go to APIs & Services → Credentials → Create Credentials → API Key. Copy the key shown.
